# 🚗 ROS 2 Autonomous Driving Foundations

This repository demonstrates the step-by-step development of an autonomous vehicle communication architecture using **ROS 2 (Robot Operating System)**. It showcases the modular node-based system used in the defense and automotive industries.

## 📌 Project Architecture
- **Vehicle Brain:** The central node that processes sensor data and makes driving decisions.
- **Sensor Publisher:** Simulates hardware sensors (e.g., LiDAR, Camera) publishing real-time environmental data.

## 📅 Development Roadmap (Daily Commits)
- [x] **Day 1:** Project setup, ROS 2 workspace architecture, and basic `vehicle_brain` node initialization.
- [x] **Day 2:** Implemented `sensor_publisher` node to broadcast simulated obstacle distances.
- [ ] **Day 3:** Implementing Subscriber logic for inter-node communication.
- [ ] **Day 4:** Custom ROS 2 Messages (`.msg`).
- [ ] **Day 5:** Parameter Server configuration for speed limits.
